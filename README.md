# craft-tap

HTTP RESTful Element API Plugin for Craft CMS

### Routes

Verb      | Path                | Action
:-------- | :------------------ | :------
GET       | /tap/{element}      | index
POST      | /tap/{element}      | store
GET       | /tap/{element}/{id} | show
PUT,PATCH | /tap/{element}/{id} | update
DELETE    | /tap/{element}/{id} | destroy
